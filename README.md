# Сайт психолога — Мария Егорова

Статический одностраничный сайт-визитка (HTML, CSS, JS).  
Адаптивная вёрстка, тёплая цветовая гамма.

## Деплой

Инструкция по выкладке на хостинг reg.ru через GitHub Actions: **[DEPLOY.md](DEPLOY.md)**.

## Локальный просмотр

Откройте в браузере файл `index.html` или поднимите локальный сервер, например:

```bash
python -m http.server 8000
```

Сайт будет доступен по адресу: http://localhost:8000
