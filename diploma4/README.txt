Документ с несколькими страницами. Положите сюда PNG по порядку:
1.png — первая страница (она же превью на сайте)
2.png — вторая страница
3.png — третья страница
и т.д.

В index.html в блоке «Документ 4» в атрибуте data-images перечислите все файлы, например:
data-images='["diploma4/1.png","diploma4/2.png","diploma4/3.png"]'
Если добавите 4.png, 5.png — допишите их в этот список через запятую.
